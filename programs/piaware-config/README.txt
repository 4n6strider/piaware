

fa_adept command line tool

aka piaware-config

This allows the user name to be set using

    fa_adept -user username

It also allowed the password to be set using

    fa_adept -password

(fa_adept interactively requests the password when this option is specified)

Options can be combined, as in

    fa_adept -user username -password

Help can be gotten with

    fa_adept -help

You can start sending from your FlightAware-equipped dump1090 to FlightAware using:

    fa_adept start

You can stop sending using

    fa_adept stop

You can restart the client using

    fa_adept restart

And you can check on the status of the client using

    fa_adept status
